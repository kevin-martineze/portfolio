# Portfolio Kevin Martinez — Plan de Implementación

## Stack
- Next.js 16 + TypeScript + Tailwind CSS v4
- shadcn/ui (componentes UI)
- Framer Motion (animaciones)
- Lucide React (iconos)

## Secciones
1. **Hero** — Nombre animado, gradient text, CTA buttons
2. **About** — Bio + tech stack badges
3. **Projects** — Cards con glassmorphismo + 3D hover
4. **Skills** — Grid de iconos con glow hover
5. **Experience** — Timeline vertical animado
6. **Contact** — Formulario + social links

- Dark mode por defecto
- Paleta: Negro `#0a0a0a` + Violeta `#8b5cf6` → Cyan `#06b6d4`
- Tipografía: Geist Sans + Geist Mono
- Efectos: Glassmorphismo, gradient borders, glow, 3D tilt
